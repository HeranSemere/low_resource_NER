# Sample NER Prompts for Low-Resource African Languages

This repository provides **example prompts** used in the study:

*"Can LLMs be used for Named Entity Recognition in Low-Resource African Languages?"*

It includes:

- `zero_shot_sample.txt` — Zero-shot NER prompt (Setswana)  
- `few_shot_sample.txt` — Few-shot NER prompt (Setswana)  
- `ensemble_sample.txt` — Ensemble / Chain-of-Thought verification prompt (Amharic)  

These prompts can be adapted for other languages or LLM experiments.
